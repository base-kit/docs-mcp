# Vitest 文档路径速查

path 前缀: `guide/`、`config/`、`api/`

| 场景 | path |
|------|------|
| **配置总览（defineConfig / test 字段 / 与 vite 集成 / `/// reference`）** | `config/index` |
| 入门 | `guide/index` |
| CLI 命令行 | `guide/cli` |
| 功能特性 | `guide/features` |
| 测试环境（jsdom/node） | `guide/environment` |
| Mock 函数 | `guide/learn/mock-functions` |
| Mock 模块 | `guide/mocking/modules` |
| 覆盖率 | `guide/coverage` |
| 调试 | `guide/debugging` |
| 快照 | `guide/snapshot` |
| Vitest UI | `guide/ui` |
| 迁移指南 | `guide/migration` |
| 配置项参考 | `config/<option>`（如 `config/environment`、`config/coverage`、`config/alias`） |
| API | `api/advanced/vitest` |
